## [1.0.0] - 2026-XX-XX
### Added
- Initial release of Japan Law MCP
- `search_legislation` tool for full-text search across all Japanese statutes (Japanese + English)
- `get_provision` tool for retrieving specific articles/provisions
- `get_provision_eu_basis` tool for EU cross-references (APPI-GDPR adequacy context)
- `validate_citation` tool for legal citation validation
- `check_statute_currency` tool for checking statute amendment status
- `list_laws` tool for browsing available legislation
- Coverage of APPI (2020 amendments), Cybersecurity Basic Act, Companies Act, Telecommunications Business Act, Unauthorized Access Act, My Number Act
- Official English translations from Japanese Law Translation portal (MOJ)
- Contract tests with 12 golden test cases
- Drift detection with 6 stable provision anchors
- Health and version endpoints
- Vercel deployment (dual tier bundled free)
- npm package with stdio transport
- MCP Registry publishing
